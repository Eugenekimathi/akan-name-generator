# akan-name-generator
akan names generator

## Description
A web application that calculates your Akan name based on your birthday and gender. Akan names are traditional Ghanaian names given to children based on the day of the week they were born.

## Author
Kimathi.E

## Setup Instructions
1. Clone this repository
2. Open `index.html` in your browser
3. Enter your birth date and gender
4. Click "Get My Akan Name" to discover your Akan name!

## live Site 

## BDD (Behavior-Driven Development)
- **Input:** User enters day, month, year, and selects gender
- **Output:** Application displays the corresponding Akan name and day of the week
- **Validation:** Alerts user if date is invalid (day: 1-31, month: 1-12)
